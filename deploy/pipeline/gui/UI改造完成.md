### 🚀 使用说明

1. **启动UI**
   ```bash
   cd c:\paddleDetection\PaddleDetection-release-2.5\PaddleDetection-release-2.5\deploy\pipeline\gui
   python main.py
   ```

2. **主题切换**
   - 点击右上角 🌓 按钮切换暗色/亮色主题
   - 暗色主题适合长时间监控（护眼）
   - 亮色主题适合明亮环境

3. **检测框控制**
   - 点击 🟩检测框 按钮开启检测框绘制
   - 按钮变为 🟥检测框 表示已开启
   - 再次点击关闭检测框

4. **统计卡片**
   - 总帧数: 累计处理的视频帧数
   - 总检测数: 累计检测到的目标总数
   - 平均FPS: 实时帧率
   - 延迟: 系统处理延迟（待实现）

5. **结果列表**
   - 显示检测结果的详细信息
   - 支持选中复制（Ctrl+C）
   - 点击 📋复制 按钮复制所有结果
   - 点击 🗑️清空 按钮清空列表
